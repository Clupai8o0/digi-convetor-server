Converting Binary to Decimal

The binary number system is a positional system where each binary digit (bit) carries a certain weight based on its position relative to the LSB. Any binary number can be converted to its decimal equivalent simple by summing together the weights of the various positions in the binary number which contain a 1. Take 110112 for example

The procedure is to find the weights (i.e., the powers of 2) for each bit position that contains a 1, and then add them up. Also note that the MSB has a weight of 24 even though its the fifth bit; this is because the LSB is the first bit and has a weight of 20.
